# ⚔️ 勇者挂机传说

一款基于 HTML/CSS/JavaScript 的放置类挂机 RPG 游戏。无需安装，直接在浏览器中打开 `index.html` 即可游玩。

---

## 🎮 游戏简介

你扮演一名勇者，通过不断击败怪物来成长变强。游戏支持自动挂机战斗，即使切换到其他标签页，战斗也会在后台继续进行。探索 8 个风格迥异的区域，挑战强大的 BOSS，收集稀有宝物和高阶技能书，学习强大的魔法技能，打造属于你的最强勇者！

---

## 🚀 快速开始

1. 直接用浏览器打开 `index.html` 文件
2. 点击 **"开始挂机"** 按钮即可自动战斗
3. 点击 **"🏰 主城"** 前往 NPC 处学习技能和精神修炼
4. 也可以点击 **"手动攻击"** 体验即时战斗快感

## 📋 核心玩法

### 战斗系统

| 操作 | 说明 |
|------|------|
| 开始挂机 | 开启自动战斗，勇者会自动攻击敌人并释放技能 |
| 手动攻击 | 立即发动一次攻击（敌人会在 0.4 秒后反击） |
| 恢复生命 | 花费金币（最大生命的 10%）将生命值回满 |
| 释放技能 | 点击技能按钮消耗魔力释放技能（有冷却时间） |

**伤害公式**：
- 基础伤害 = max(1, 攻击力 - 敌人防御 × 0.5)
- 暴击时伤害 × 暴击伤害倍率
- 攻速达到阈值后，每次攻击伤害会获得倍数加成

**死亡惩罚**：
- 普通战斗：损失 50~250 金币，30% 概率丢失一件未锁定的宝物
- BOSS 战：损失 50~250 金币，BOSS 逃走需重新收集线索

### 区域系统

游戏共有 **8 个区域**，每个区域有不同的等级要求和难度倍率：

| 区域 | 等级要求 | 难度倍率 | 宝物掉率 | 技能书掉率 | 备注 |
|------|---------|---------|---------|-----------|------|
| 🌲 新手村 | 1 | ×1 | 0% | 0% | 初始区域 |
| 🌲 暗黑森林 | 5 | ×2 | 0% | 0% | |
| 🌋 熔岩洞穴 | 10 | ×3 | 0% | 2% | |
| ❄️ 冰雪山脉 | 15 | ×4 | 3% | 4% | 开始掉落宝物和技能书 |
| 🏰 恶魔城堡 | 20 | ×5 | 8% | 6% | |
| 🐉 龙之巢穴 | 30 | ×7 | 12% | 8% | BOSS：龙王 |
| ☁️ 天空之城 | 40 | ×10 | 18% | 10% | BOSS：天空领主 |
| 🔥 地狱深渊 | 50 | ×15 | 25% | 12% | BOSS：深渊魔王 |

> **注意**：进入 BOSS 区域需要先击败前一个 BOSS 才能解锁下一区域。

### BOSS 挑战

- 在 BOSS 区域击败小怪有 **15%** 概率掉落线索
- 收集足够线索后可挑战 BOSS
- BOSS 被击败后该区域永久解锁更高难度

| BOSS | 所需线索 | 击败奖励 |
|------|---------|---------|
| 🐉 龙王 | 10 | 必掉史诗及以上宝物 |
| ☁️ 天空领主 | 15 | 必掉史诗及以上宝物 |
| 👿 深渊魔王 | 35 | 必掉史诗及以上宝物 |

## 🔮 技能系统

### 新属性：精神与魔力

| 属性 | 说明 | 获取方式 |
|------|------|---------|
| 🧠 精神 | 影响技能伤害倍率和魔力上限 | 升级+1、精神修炼+2 |
| 💧 魔力 | 释放技能消耗的资源 | 战斗外自动恢复 |

**魔力规则**：
- 最大魔力 = 50 + 精神 × 3 + 等级 × 2
- 战斗外每 2 秒恢复 5% 最大魔力
- 每次普通攻击恢复 1 点魔力
- 升级时魔力回满

### 技能分类

#### 基础技能（主城 NPC 直接学习）

| 技能 | 元素 | 类型 | 效果 | 学习费用 |

| 🔥 火球术 | 火 | 伤害 | 单体魔法伤害 | 100 金币 |
| ❄️ 冰箭 | 冰 | 伤害 | 单体魔法伤害 | 100 金币 |

| 🛡️ 护盾术 | 圣 | 增益 | 临时防御加成（10秒） | 120 金币 |

### 技能升级

- 基础技能最高 **10 级**，高阶技能最高 **5 级**
- 每次升级伤害提升 **15%**
- 升级费用 = 基础费用 × 1.3^(等级-1)

### 技能书获取与学习

1. **掉落**：击败怪物有几率掉落技能书（第 3 关起）
2. **鉴定**：在主轴 NPC 处花费金币鉴定未鉴定的技能书
3. **学习**：鉴定后可在 NPC 处学习该技能
4. **出售**：不需要的技能书可以出售换取金币

## 🎒 宝物系统

从第 4 关（冰雪山脉）开始，击败怪物有几率掉落宝物。宝物分为 4 个稀有度：

| 稀有度 | 颜色 | 权重 | 说明 |
|--------|------|------|------|
| 普通 | 灰色 | 55% | 基础属性加成 |
| 稀有 | 蓝色 | 30% | 中等属性加成 |
| 史诗 | 紫色 | 12% | 强力属性加成 |
| 传说 | 橙色 | 3% | 顶级属性加成 |

### 宝物属性类型

- **攻击** / **防御** / **生命**：基础三维属性
- **暴击率** / **暴击伤害**：提升输出上限
- **经验加成** / **金币加成**：提升资源获取效率
- **破甲（固定值）**：无视敌人固定数值的防御
- **破甲（百分比）**：无视敌人百分比防御（上限 35%）

### 宝物操作

| 功能 | 说明 |
|------|------|
| 🔨 强化 | 消耗 1 个同名宝物 + 金币，提升宝物等级（属性 +50%/级） |
| 💰 出售 | 获得金币，锁定的宝物无法出售 |
| 🔒 锁定 | 点击 🔓 锁定宝物，防止误操作出售或死亡掉落 |
| 筛选 | 可按属性类型筛选显示宝物 |
| 自动强化 | 自动消耗金币和多余宝物进行强化（需开启开关） |

## 💾 存档管理

| 功能 | 说明 |
|------|------|
| 保存进度 | 将当前进度存入浏览器本地存储 |
| 读取进度 | 从本地存储恢复进度 |
| 导出存档 | 将存档编码为 Base64 字符串，可复制到剪贴板 |
| 导入存档 | 粘贴 Base64 存档代码恢复进度 |
| 重置进度 | 清空所有数据，重新开始 |

> 游戏每 30 秒会自动静默保存一次。关闭页面时也会自动保存。
> **旧存档兼容**：读取旧版本存档时会自动补充新属性（精神、魔力等）的默认值。

## 🛠️ 技术说明

### 浏览器后台运行优化

游戏针对浏览器标签页切换做了特殊优化：

- 当页面切换到后台时，浏览器的 `setInterval` 会被节流到约 1000ms
- 游戏通过记录时间戳，在切回前台时批量计算错过的攻击次数和技能释放
- 最多批量处理 500 次攻击，确保长时间挂机不会丢失进度

### 文件结构

```
.
├── index.html        # 游戏主页面（HTML 结构）
├── css/
│   └── style.css     # 游戏样式
├── js/
│   └── game.js       # 游戏逻辑
└── README.md         # 本说明文档
```

### 技术栈

- 纯原生 HTML5 / CSS3 / JavaScript（ES6+）
- 无需任何外部依赖
- 使用 `localStorage` 本地持久化存档
- 响应式布局，支持移动端浏览
